Dashboard

Trello dashboard creator

This tool will take project boards created on Trello and turn them into printable dashboards.

The tool has been designed to work on a modern desktop browser, such as Apple Safari, Google Chrome or Firefox, and is built using HTML5 components. Whilst the app may work on other devices, it is recommended you use it on a desktop or laptop device.

You will need to authorise this web app to access your Trello account to continue.
